About Meфыфыфы

Здравствуйте, меня зовут Кондратенко Максим. Я занимаюсь версткой сайтов. В своей работе я использую сборщик проектов Gulp для автоматизации трудоёмких задач, повторяющихся в процессе работы. Это значительно ускоряет процесс разработки. Прописываю стили в препроцессоре sass. Для адаптивноcти сайта применяю как библеотеку Bootstrap, так и css медиа запросы для точного позиционирования элементов сайта на разных размерах экрана. Работаю с svg форматом. Использую различные плагины, или пишу сам.

В дальнейшем планирую углубленно изучать JavaScript, популярные библиотеки и фреймворки для него, такие как: Vue, Angular, React. И также выучить CMS.
